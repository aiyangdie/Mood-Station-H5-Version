# 心情驿站 H5 版本

本项目为心情驿站（H5移动端版本），请勿删除本地代码，便于多端维护。

## 主仓库

本项目主仓库地址：[https://github.com/aiyangdie/-Xin-Qing-Yi-Zhan](https://github.com/aiyangdie/-Xin-Qing-Yi-Zhan)

## 部署说明

1. 将 inde 文件夹下所有内容上传至服务器或静态空间。
2. 访问 inde/index.html 即可使用。
3. 如需部署到 GitHub Pages：
   - 新建一个仓库（如 mood-h5），上传 inde 文件夹下所有内容到仓库根目录。
   - 在仓库设置中启用 GitHub Pages，选择 main 分支和根目录。
   - 稍等片刻，即可通过 pages 提供的链接访问。

## 说明
- 本版本为纯静态 HTML/CSS/JS，无需后端。
- 如需同步主仓库内容，请参考主仓库文档。
- 如有问题欢迎在主仓库提交 issue。
